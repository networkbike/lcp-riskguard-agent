# Assets

This folder is reserved for screenshots, sample outputs, or
diagrams that Anvita Flow's Developer Console may use to populate
the Agent Card.

Recommended content (add before publishing):

- `screenshot-success.png` — a screenshot of the runner returning
  `band: HEALTHY` for `native:PROS mainnet`.
- `screenshot-watch.png` — a screenshot of the runner returning
  `band: WATCH` for a known-stress test token on Atlantic.
- `flow-diagram.png` — the ASCII flow diagram from `SKILL.md`
  rendered as a PNG.

The folder ships empty by default; populate it before submitting
to Anvita Flow.
