How this works 
This UI calls the same runner you'd invoke from a terminal:
  LCP_TARGET=<target> LCP_NETWORK=<network> LCP_THRESHOLD=<threshold> bash scripts/run.sh

The runner is hosted in this static page. We make a fetch() call
to a public Pharos RPC for the live signals and the LCP math is
re-implemented in JS for the demo. The "real" Agent on Anvita Flow
calls the same Skill via cast / forge.

Result

Click "Run LCP RiskGuard" to score the default target (native:PROS on Pharos mainnet).

Why LCP RiskGuard

🔍
Reproducible
Every score is byte-for-byte reproducible from public RPC data. No ML, no oracles, no off-chain enrichment.
🛡️
Read-only by design
No wallet, no signing, no transaction broadcast. Refuses PRIVATE_KEY. Three layers of defense.
⚡
Fast
~8 seconds end-to-end on Pharos mainnet public RPCs. Stateless, idempotent, parallelizable.
🎯
Actionable
Three-band output (HEALTHY/WATCH/CRITICAL), 0-100 score, top 3 drivers. Designed to drive alerts, not just dashboards.
🧩
Composable
Wraps one Skill today. Designed to compose with future Skills (gas oracle, price feed, exit-plan).
📞
Free during beta
No payment module yet. The Agent is free on Anvita Flow while the pricing beta runs.
